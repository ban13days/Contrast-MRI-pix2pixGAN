# V-MAP 프로젝트 QA 통합 체크리스트

## 1. 데이터 인입 및 전처리
- [v] 업로드된 다중 MRI 이미지(단일 환자 스캔 세트)의 총 슬라이스 수 누락 없이 식별
- [v] 인입된 정상 슬라이스 대상 명암비 개선 및 노이즈 제어(CLAHE) 전처리 알고리즘 정상 적용 
- [v] 정상 슬라이스 대상 명암비 개선 및 노이즈 제어 알고리즘 정상 적용 확인

## 2. AI 추론 및 성능 검증
- [v] 전처리 이미지를 모델에 입력하여 종양 영역(ROI) 세그먼트 마스크 정상 도출 
- [v] 비조영 이미지와 예측 세그먼트 결합을 통한 가상 조영 효과(GAN) 적용 이미지 생성 정합성 확인 
- [v] 단일 환자 3D MRI 세트(약 100장 기준) 업로드부터 시각화까지 전체 AI 파이프라인 처리 60초 이내 달성 

## 3. 임상 뷰어 및 시각화 (UI/UX)
- [v] 분석 완료 후 뷰어 진입 시 상단 헤더에 환자 정보 메타데이터 정상 매핑 
- [v] 비조영 원본(좌)과 가상 조영 결과(우) 영상 쌍이 레이아웃 붕괴 없이 병렬 렌더링 
- [v] 마우스 휠 및 화살표 고속 스크롤 시 좌우 패널 슬라이스 지연 없이 1:1 실시간 동기화 
- [v] 병변 영역 표시 버튼 클릭 시, 종양 위치 좌표와 일치하는 Red 마스크 오버레이 렌더링 
- [v] 하단 통계 데이터(면적 비율 등) 레이아웃 붕괴 없이 표출 및 AI 신뢰도 게이지 바 즉각적 리렌더링 

## 4. 예외 처리 및 데이터 반출
- [v] 종양 미감지 슬라이스로 이동 시, 통계 데이터가 잔존하지 않고 0으로 안전하게 초기화(Reset)되는지 검증 
- [v] 현재 렌더링된 가상 조영 결과물 및 마스크 데이터의 누락 없는 단일 압축(Zip) 파일 다운로드 확인 
- [v] 외부 반출용 다운로드 이미지 내 환자의 식별 가능 정보(PHI) 완전 삭제 처리 확인
